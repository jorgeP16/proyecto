¿QUE ES PROGRAMACION?

La programación es el proceso de crear un conjunto de instrucciones que le dicen a una computadora como realizar algún tipo de tarea. Pero no solo la acción de escribir un código para que la computadora o el software lo ejecute. Incluye, además, todas las tareas necesarias para que el código funcione correctamente y cumpla el objetivo para el cual se escribió

En la actualidad, la noción de programación se encuentra muy asociada a la creación de aplicaciones de informática y videojuegos. En este sentido, es el proceso por el cual una persona desarrolla un programa, valiéndose de una herramienta que le permita escribir el código (el cual puede estar en uno o varios lenguajes, como C++, Java y Python, entre muchos otros) y de otra que sea capaz de “traducirlo” a lo que se conoce como lenguaje de máquina, que puede "comprender" el microprocesador.

¿PARA QUE NOS SIRVE LA PROGRAMACION?

Para hacer que los softwares que hacen nuestras vidas más fáciles, puedan funcionar. Por ejemplo, Google Chrome, es un software que nos hace el uso de internet más sencillo, ¿imaginas cómo serían tus búsquedas, suscripciones, archivar en la nube sin él? Gracias a los softwares, funcionan nuestros teléfonos móviles, computadoras, los cajeros automáticos, las aplicaciones, la vida sin ellos sería menos entretenida. Los juegos y la creatividad también pueden estimularse gracias a programas como scratch, por ejemplo.

¿CUAL ES LA IMPORTANCIA DE LA PROGRAMACION?

Ya que sabes para qué nos sirve la programación, conocer su importancia se hace más sencillo. Gracias a la programación, los softwares funcionan y hacen que los trabajos se ejecuten de manera más eficiente. Por ejemplo, el tratar de optimizar un proceso, es una de las actividades para la que nos sirve la programación dinámica o tratar de crear un software para disminuir costos que es para lo que nos sirve la programación lineal. Existen otros tipos de programación como la que está orientada a objetos, ¿y para que nos sirve la programación orientada a objetos? Pues está orientada a disminuir errores y a la reutilización de los códigos fuente. La programación es importante porque: Permite que los softwares funcionen. Sin los softwares funcionando, el hardware sería una creación inútil. Permite hacer tareas de forma más rápida. Permite resolver problemas de manera más eficiente. Gracias a ella, se pueden tener herramientas para hacer cálculos que de otra forma sería imposible para el ser humano. En el mundo civilizado de hoy, es casi impensable vivir sin la tecnología a base de programación informática
